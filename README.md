🚀 Features
- Random number generation (1–100)
- User-friendly interface
- Instant feedback on each guess
- Displays "Too High" or "Too Low" hints
- Counts the number of attempts
- Automatically starts a new game after a correct guess

 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

🎮 How to Play

1. Enter a number between **1 and 100**.
2. Click the **Guess** button.
3. The game will display:
   - 📉 Too Low!
   - 📈 Too High!
   - 🎉 Congratulations! (if correct)
4. The game resets automatically after the correct guess.

- Random Number Generation
- DOM Manipulation
